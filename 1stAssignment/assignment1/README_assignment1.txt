ΑΣΚΗΣΗ 1

ΔΙΑΔΙΚΑΣΙΑ ΜΕΤΑΦΡΑΣΗΣ

Το αρχείο δοκιμής θα πρέπει να έχει την ονομασία:	input

Αρχικά, μεταγλωττίζεται ο κώδικας με την εντολή: 	make
Στο makefile που το πρόγραμμα μεταγλωττίζεται με fsanitize address για έλεγχο leaks και -lpthread για τη σύνδεση της βιβλιοθήκης pthread.

Το πρόγραμμα εκτελείται με την εντολή: 			./main 64
(όπου 64 το μέγεθος του buffer)

ΔΟΚΙΜΗ

Για να έχει εκτελεστεί σωστά το πρόγραμμα θα πρέπει τα 3 files: input, input.copy, input.copy2 να είναι ίδια. Αυτό ελέγχθηκε με τις εντολές: diff input input.copy, diff input.copy input.copy2 και diff input input.copy2.

Παράλληλα, για να επιβεβαιωθεί η ταυτόχρονη λειτουργία των threads χρησιμοποιήθηκαν print statements με χρώμα κίτρινο για τις εκτελέσεις των εντολών write και πράσινο για τις εντολές read, ώστε να παρατηρηθεί ότι η εκτέλεση των εντολών δεν είναι σειριακή/γραμμική. 

Το πρόγραμμα δοκιμάστηκε με διάφορα μεγέθη αρχείου. Από αυτά παραδόθηκαν τρία ενδεικτικά αρχεία που ελέγχουν τη λειτουργία για μεγέθη μικρότερα και μεγαλύτερα του μεγέθους του ring buffer (12 bytes, 200 bytes και 10000 bytes).

Από τις εκτελέσεις παρατηρούμε ότι οι χαρακτήρες γράφονται στα σωστά σημεία του buffer και οι read και write εντολές εναλλάσσονται με τυχαία σειρά και με διαφορετικό τρόπο σε διαφορετικές εκτελέσεις.
